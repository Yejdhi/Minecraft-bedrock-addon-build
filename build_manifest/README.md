# 关于此网页工具
### 当前版本
最新的版本号为2.0.1beta
- 无后缀-正式版
- pre-预览版
- beta-测试版
### 为什么需要它？
每次做新的附加包都需要从其他包重新复制一份manifest.json来修改做使用，而且有时作者也并不清楚自己的包到底适用于什么样的版本，或者出现某些错误。
所以这时候我们就需要一个能够自定义，且方便用于创建新文件的工具。
### 如何使用它？
- 打开这个网页文件，填写其中所需要的信息。
- 点击绿色预览按钮可预览修改。
- 点击下载按钮可打包并保存为json文件到本地。
